## 残疾人社会工作的主要内容

### 残疾人社会工作服务理念的变迁 <Badge text="了解" type="tip" />

- 防止把导致残疾的原因归因到人体的生理性因素或道德层面的缺失
- 防止把残疾人个体的问题“普世化”
- 防止把残疾人简单地看作是社会的“受害者”
- 防止把残疾人看作“理所当然”需要帮助和社会支持的人

### 残疾人社会工作服务模式的变迁 <Badge text="必考" type="warning" />

::: tabs

@tab 工作理论层面

致残原因的理论分析：从个人责任转向社会责任

@tab 工作方法层面

工作方法理念：由供养理论向回归社会理论转变

直接接入模式：由单一个案向综合服务模式转变
